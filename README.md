# ECHO Connect 🚀
- ECHO Connect é uma **rede social** focada em **programadores**, permitindo a publicação e o compartilhamento de **estudos** e conteúdos educativos para a comunidade de desenvolvedores. A plataforma visa ajudar programadores de todos os níveis a se conectarem e aprenderem juntos. 💻✨

## Funcionalidades 🎯

- **Criação de Conta**: Cadastro e login para usuários.
- **Publicação de Postagens**: Usuários podem criar posts com conteúdo educativo.
- **Sistema de Likes**: Interaja com posts através de curtidas. ❤️
- **Comente nos Posts**: Deixe comentários em postagens de outros usuários. 💬
- **Tags**: Adicione tags aos posts para facilitar a busca por temas. 🏷️
- **Perfil de Usuário**: Visualize e edite seu perfil. 🖼️

## Tecnologias Utilizadas 🛠️
> - ``Javascript``: Para funcionalidades da interface
> - ``CSS3``: Para estilização das páginas
> - ``Node.js``: Ambiente de execução JavaScript no lado do servidor.
> - ``Express``: Framework web para Node.js.
> - ``MongoDB``: Banco de dados NoSQL para armazenar dados de usuários e posts.
> - ``Handlebars``: Motor de templates para renderizar HTML no lado do servidor.
